# Expert Tree Services

A responsive static website mockup for a tree-services company serving the Wilmington, Delaware area.

## Project structure

- `index.html` — homepage and page content
- `assets/` — stylesheets, JavaScript, and responsive images

## Preview locally

From this directory, run:

```bash
python3 -m http.server 3000
```

Then open <http://localhost:3000/>.

## GitHub Pages

This repository is ready to publish from the `main` branch and repository root. In GitHub, open **Settings → Pages**, select **Deploy from a branch**, then choose `main` and `/(root)`.

## Before production launch

- Replace sample testimonials with verified customer reviews.
- Verify all licensing, insurance, certification, service-area, address, and operating-hours details.
- Connect the estimate form to a secure server-side form handler.
- Keep API keys and other secrets on the server; never place them in `index.html` or browser JavaScript.
